# Coin Telemedicina Web

**Coin Telemedicina Web** is the web version of a multi-role telemedicine platform designed to connect **patients, doctors, and medical interpreters** through a streamlined digital healthcare experience.

The platform enables users to **book medical appointments, manage doctor availability, assign interpreters for language support, and join scheduled video consultations** through a clean and accessible web interface.

## Key Modules

### Patient Portal
- Browse available doctors
- Book appointments based on consultation time slots
- Select a **medical interpreter** during appointment scheduling when required
- View upcoming and previous appointments
- Join scheduled online video consultations

### Doctor Portal
- Set and manage available consultation timings
- Review booked appointments
- Conduct scheduled video consultations with patients
- Attend interpreter-assisted sessions when language support is included

### Medical Interpreter Portal
- View assigned interpretation sessions
- Join scheduled consultations as a third participant
- Help facilitate clear communication between doctor and patient
- Support multilingual telemedicine workflows

## Key Features

### Online Appointment Booking
- Patients can schedule consultations directly through the web platform
- Booking flow is connected with doctor availability
- Appointments are organized and easy to manage

### Doctor Availability Management
- Doctors can define their available days and time slots
- Patients can only book from open consultation timings
- Helps maintain a structured scheduling process

### Interpreter-Assisted Healthcare
- Patients can select a **medical interpreter** when booking an appointment
- The assigned interpreter becomes part of the consultation workflow
- Designed to improve healthcare accessibility across language barriers

### Scheduled Video Consultations
- Patients and doctors can join video calls at the scheduled time
- Medical interpreters can join the same consultation when selected
- Supports complete remote care sessions within the platform

### Role-Based Web Experience
- Dedicated portals for:
  - Patients
  - Doctors
  - Medical Interpreters
- Each user type receives a purpose-built interface and workflow

## Technology Stack

- **Flutter Web & Dart** — Cross-platform web application development
- **Real-Time Video Consultation Integration** — Scheduled multi-participant calls
- **Role-Based Portal System** — Separate workflows for patients, doctors, and interpreters
- **Appointment Scheduling Module** — Availability, booking, and consultation management
- **Interpreter Assignment Flow** — Language support within telemedicine sessions

## Project Highlights

- Full web-based telemedicine platform
- Patient, doctor, and medical interpreter portals
- Doctor time-slot and appointment management
- Optional interpreter selection during booking
- Three-party scheduled video consultation flow
- Designed for accessible and multilingual remote healthcare

## Project Overview

Coin Telemedicina Web was developed as a browser-based extension of the telemedicine ecosystem, allowing patients, doctors, and medical interpreters to manage healthcare consultations through a professional and accessible web platform.

This project demonstrates expertise in building **role-based healthcare systems, appointment booking workflows, interpreter-assisted communication, and real-time telemedicine web solutions**.
